# Lung Cancer Prediction using Machine Learning

## Overview
This project focuses on predicting the presence of lung cancer using biomedical data.
Multiple machine learning and nature-inspired algorithms were implemented and evaluated.

## Implementation
The complete implementation was developed using Google Colab.

Notebook: LungCancer.ipynb

## Algorithms Used
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Random Forest
- Naive Bayes
- Nature-inspired optimization techniques

## Evaluation Metrics
- Accuracy
- Precision
- Sensitivity
- F1-score

## Tools & Libraries
- Python
- NumPy, Pandas
- Scikit-learn
- Matplotlib

## Notes
This project was completed as a minor academic project and implemented using Google Colab.
